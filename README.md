
# Customer Behavior Analysis - Transportation Industry

## Project Overview

This project analyzes customer behavior and transportation service performance using SQL, Python, and Power BI.

The objective is to identify customer spending patterns, top-performing transportation services, revenue by category, and key business KPIs.

---

## Tools Used

* SQL (Oracle)
* Python (Pandas, Matplotlib)
* Power BI
* GitHub

---

## Database Structure

Tables used:

* CLIENTES
* CATEGORIAS
* PRODUCTOS
* ORDENES
* DETALLE_ORDEN

Relationships were modeled using primary and foreign keys to simulate a transportation and logistics business.

---

## Key Performance Indicators (KPIs)

* Total Revenue
* Total Orders
* Total Customers
* Average Order Value (Ticket Average)

---

## Analysis Performed

### Top Customers by Revenue

Identified the customers generating the highest revenue.

### Revenue by Category

Analyzed which transportation categories produce the highest income.

### Revenue by Payment Method

Compared revenue generated by different payment methods.

### Service Performance

Evaluated the most requested transportation services.

---

## Dashboard

The Power BI dashboard includes:

* Interactive filters
* KPI cards
* Customer ranking
* Revenue by category
* Transportation-themed design

---

## Main Findings

* National Shipping generated the highest revenue.
* Luis Castillo was the highest spending customer.
* Average order value exceeded Q4,000.
* Transportation services showed clear differences in profitability.
